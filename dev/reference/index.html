WeightNormalizedRBMs.gu2w — Method
gu2w(g, u) -> w, un

Returns w, un (as a NamedTuple), from g, u, where:

\[\mathbf{w} = g \frac{\mathbf{u}}{\|\mathbf{u}\|}\]

and un are the norms $\|\mathbf{u}\|$.

source
WeightNormalizedRBMs.w2gu — Method
w2gu(w, un) -> g, u

Returns g, u (as a NamedTuple), such that

\[\mathbf{w} = g \frac{\mathbf{u}}{\|\mathbf{u}\|}\]

where the norms $\|\mathbf{u}\|$ are given by un.

WeightNormalizedRBMs.∂wnorm — Method
∂wnorm(∂w, g, u)

Given the gradients ∂w of a function f(w) with respect to w, returns the gradients ∂g, ∂u of f with respect to the re-parameterization:

\[\mathbf{w} = g \frac{\mathbf{u}}{\|\mathbf{u}\|}\]
